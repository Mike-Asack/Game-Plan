## 01. 게임 컨셉 및 개요

### 1.1 게임 컨셉

**카드의 높낮이를 예측하는 확률 기반 카지노 게임**

- **장르**: 카지노 인스턴트 / 확률 예측 게임
- **플랫폼**: 웹 기반 (모바일/데스크톱 반응형)
- **게임 방식**: 턴제 예측 게임
- **핵심 메커니즘**: 다음 카드가 현재 카드보다 높을지(HIGH) 낮을지(LOW) 예측

### 1.2 게임 목표

- 연속으로 올바른 예측을 하여 **Multiplier를 증가**시킴
- 적절한 타이밍에 **CASH OUT**하여 최대 수익 획득
- 무한 덱 시스템으로 **공정한 확률** 보장

### 1.3 게임 흐름 요약

```
베팅 설정 → BET 클릭 → 카드 등장
→ HIGH/LOW 선택 → 결과 확인
→ 성공 시 계속 or CASH OUT
→ 실패 시 게임 종료
```

## 02. 플레이 루프

### 2.1 게임 시작 루프

```
1. 초기 상태
   ├─ Balance 표시
   ├─ Bet Amount 설정 (기본: $10)
   └─ BET 버튼 활성화

2. BET 버튼 클릭
   ├─ Balance에서 Bet Amount 차감
   ├─ 첫 번째 카드(Current) 랜덤 생성
   ├─ Streak (연승) = 0, Multiplier = 1.00x 초기화
   └─ HIGH/LOW 버튼 활성화
```

### 2.2 핵심 플레이 루프

```
┌─────────────────────────────────┐
│  1. 현재 카드 확인               │
│     ├─ 카드 랭크 파악           │
│     ├─ HIGH/LOW 배당률 확인     │
│     └─ 전략 수립                │
├─────────────────────────────────┤
│  2. [선택적] SHUFFLE 실행        │
│     └─ 현재 카드를 새 카드로 교체│
├─────────────────────────────────┤
│  3. HIGH 또는 LOW 선택           │
│     ├─ 버튼 클릭                │
│     └─ Next 카드 뒷면 표시      │
├─────────────────────────────────┤
│  4. 카드 공개 애니메이션          │
│     ├─ Flip 애니메이션 (0.6초)  │
│     └─ 결과 판정                │
├─────────────────────────────────┤
│  5-A. 성공 시                    │
│     ├─ 승리 애니메이션 (초록)    │
│     ├─ Streak (연승) +1         │
│     ├─ Multiplier × 배당률      │
│     ├─ Win 금액 업데이트         │
│     ├─ Next → Current 이동      │
│     └─ 계속 플레이 or CASH OUT  │
├─────────────────────────────────┤
│  5-B. 실패 시                    │
│     ├─ 패배 애니메이션 (빨강)    │

│     └─ 게임 초기화 (리셋)        │
└─────────────────────────────────┘
          ↑                ↓
          └────← 계속 ←─────┘
```

### 2.3 CASH OUT 루프

```
1. CASH OUT 버튼 클릭
   ├─ Win 금액을 Balance에 추가
   ├─ 승리 팝업 표시 (금액 표시)
   ├─ Cashout 사운드 재생
   └─ 게임 초기화

2. 초기 상태로 복귀
   └─ 새로운 베팅 가능
```

#### 1. 무한 덱 시스템

```javascript

```

- **장점**: 카드 카운팅 불가능, 공정한 확률
- **특징**: 같은 카드가 연속으로 나올 수 있음

#### 2. A와 K의 특수 규칙

| 카드 | HIGH 선택 시 | LOW 선택 시 |
|------|--------------|-------------|
| **A (최저)** | Same 제외, 2~K만 승리<br>(48장/52장) | **Same만 승리**<br>(4장/52장) |
| **K (최고)** | **Same만 승리**<br>(4장/52장) | Same 제외, A~Q만 승리<br>(48장/52장) |
| **2~Q** | High or Same 승리 | Low or Same 승리 |

#### 3. SHUFFLE 시스템

- 게임 중 언제든지 현재 카드 교체 가능
- 무료 기능 (추가 비용 없음)
- 불리한 카드(A, K)를 회피 전략

### 4.2 게임 규칙

#### 베팅 규칙

```
최소 베팅: $1
최대 베팅: $1,000
조정 단위: $10 (±버튼)
직접 입력: 모달에서 숫자 입력
```

#### 게임 진행 규칙

1. **베팅 필수**: Balance ≥ Bet Amount
2. **예측 필수**: HIGH 또는 LOW 중 하나 선택
3. **연속 플레이**: 성공 시 다음 턴 자동 진행
4. **실패 시 종료**: 모든 진행 상황 초기화
5. **CASH OUT 권한**: 언제든지 Win 금액 수령 가능

#### 승리 조건

```javascript
// A 카드
if (currentCard.rank === "A") {
  if (guess === "high") {
    isCorrect = newNextCard.value > 1; // 2~K
  } else {
    isCorrect = newNextCard.value === 1; // A only
  }
}

// K 카드
if (currentCard.rank === "K") {
  if (guess === "high") {
    isCorrect = newNextCard.value === 13; // K only
  } else {
    isCorrect = newNextCard.value < 13; // A~Q
  }
}

// 일반 카드 (2~Q)
if (guess === "high") {
  isCorrect = newNextCard.value >= currentCard.value;
} else {
  isCorrect = newNextCard.value <= currentCard.value;
}
```

## 05. 보상 / 수치 테이블 (TORY 환수율 및 RTP 체크 필요)

### 5.1 배당률 계산 공식

```javascript
// 확률 기반 배당 계산
function calculateOdds(currentCard, guess) {
  const totalCards = 52; // 무한 덱
  const sameRankCards = 4; // 같은 랭크 카드

  // 승리 가능한 카드 수 계산
  let winningCards = 0;

  if (currentCard.rank === "A") {
    winningCards = guess === "high" ? 48 : 4;
  } else if (currentCard.rank === "K") {
    winningCards = guess === "high" ? 4 : 48;
  } else {
    const higherCards = (13 - currentCard.value) * 4;
    const lowerCards = (currentCard.value - 1) * 4;
    winningCards = guess === "high"
      ? higherCards + sameRankCards
      : lowerCards + sameRankCards;
  }

  // 승률 계산
  const winChance = winningCards / totalCards;

  // 배당률 = (1 / 승률) × 하우스 엣지 (0.95)
  const rawPayout = (1 / winChance) * 0.95;
  const payout = Math.max(1.05, rawPayout); // 최소 1.05x 보장

  return { winChance, payout };
}
```

### 5.4 Multiplier 시스템

```javascript
// 연속 성공 시 Multiplier 누적
const newMultiplier = multiplier * payout;

// 예시 시나리오
Bet: $10

Turn 1: 7 → HIGH → 8 (성공)
  Payout: 1.77x
  Multiplier: 1.00 × 1.77 = 1.77x
  Win: $10 × 1.77 = $17.70

Turn 2: 8 → HIGH → K (성공)
  Payout: 2.06x
  Multiplier: 1.77 × 2.06 = 3.65x
  Win: $10 × 3.65 = $36.50

Turn 3: K → HIGH → K (성공)
  Payout: 12.35x
  Multiplier: 3.65 × 12.35 = 45.08x
  Win: $10 × 45.08 = $450.80

CASH OUT: Balance +$450.80 💰
```

### 5.5 하우스 엣지 분석

```
이론적 배당: 1 / 승률
실제 배당: (1 / 승률) × 0.95

하우스 엣지: 5%

예시:
- 7 카드 HIGH (53.8% 승률)
  이론적: 1 / 0.538 = 1.86x
  실제: 1.86 × 0.95 = 1.77x
  하우스 수익: 5%
```
